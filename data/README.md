# Datasets Summary

The original CSV files for each dataset were downloaded directly from their respective public sources and contain only minor modifications, specifically the removal of metadata rows and formatting elements that were not relevant to analysis. No data values were altered during this process. These cleaned CSV files were then processed and exported as Tableau .hyper files, which are the finalized versions used for data modeling and visualization development within Tableau.

## Dataset 1: Employee Count

The Employee Count dataset was sourced from Statistics Canada and contains annual employment totals by province and industry classification. This dataset provides insight into labour market size and regional employment distribution across sectors. The data was accessed on February 1, 2026, and reflects publicly released official statistics. There are no usage restrictions associated with this dataset, and it is available for academic research, analysis, and data visualization purposes.

## Dataset 2: Job Vacancy and Payroll Data

The Job Vacancy and Payroll dataset was obtained from Statistics Canada and includes monthly data on job vacancies, payroll employees, and job vacancy rates across industries categorized under the North American Industry Classification System (NAICS). This dataset supports time-series labour market analysis and industry-level comparisons. The data was accessed on February 1, 2026. It is publicly available through official government statistical releases and carries no usage restrictions, allowing it to be used for academic analysis and visualization projects.

## Dataset 3: Employee Wage Data

The Employee Wage dataset was sourced from Statistics Canada and provides wage-related information by industry and time period. This dataset allows for analysis of earnings trends, sectoral wage differences, and changes in compensation over time. The data was accessed on February 1, 2026 and is publicly available. There are no usage restrictions associated with this dataset, and it may be used for academic research, policy analysis, and visualization purposes.

## Dataset 4: Government Investment Data

The Government Investment dataset was obtained from a publicly available Canadian government source and contains information on investment levels by industry. This dataset supports analysis of how public funding is distributed across sectors and how investment patterns may relate to employment and labour market trends. The data was accessed last Thursday. It is publicly accessible and carries no usage restrictions, making it suitable for academic research, analytical reporting, and visualization development.
